# CodeAtlas Analyzer Contract

Analyzers emit facts and evidence. They do not emit final user-facing conclusions.

MVP analyzers:

- Spoon Java source analyzer.
- Spring MVC adapter.
- Struts1 adapter.
- WebAppContext + Jasper JSP analyzer.
- JetHTMLParser/Jericho tolerant JSP extractor.
- MyBatis/JSqlParser SQL analyzer.

Every emitted fact must include evidence and confidence.
